# Odion stijlgids — Design tokens

Gebaseerd op de CSS van odion.nl.

## Kleuren

| Token | Waarde | Gebruik |
|-------|--------|---------|
| Primary | `#762283` | Knoppen, links, accenten |
| Primary dark | `#641d6f` | Hover states |
| Accent | `#ffff64` | Highlights, badges |
| Surface | `#f8f9fa` | Achtergronden, cards |
| Text | `#000` | Body text |
| Text light | `#fff` | Op paarse achtergrond |

## Typografie

| Element | Font |
|---------|------|
| Body | Montserrat, Helvetica, Arial, sans-serif |
| Koppen (optioneel) | Swift Neue, Cambria, Times New Roman, serif |

## Vormen

| Element | Radius |
|---------|--------|
| Cards, inputs | 0.25rem (4px) |
| Buttons, tags | 3.125rem (50px / pill) |
